基本文字格式部分

1.全局样式 style.css:

  • 移除body的margin声明
  • 设置body的背景色为白色
  • 为排版设置了基本的字体、字号和行高
  • 设置全局链接颜色，且当链接处于悬浮：“hover”状态时才会显示下划线样式

2.代码风格

  • code 标签
    一般是针对于单个单词或单个句子的代码
  • pre 标签
    一般是针对于多行代码（也就是成块的代码）
  • 不管使用那种代码风格，在代码中碰到小于号（<）要使用硬编码 &lt;来代替,大于号（>）使用&gt;来代替
  • .pre-scrollable 类名
    只要在pre标签类名上添加类名.pre-scrollable,就可以控制代码块区域最大高340px，一旦超出这个高度，就会在Y轴
    出现滚动条。

3.标题

1.重置了h1-h6的默认样式

  • 重新设定了margin-top和margin-bottom的值， h1-h3重置后的值都是20px;
    *h4-h6重置后的值都是10px；
  • 所有的标题行高都是1.1（也就是font-size的1.1倍） 而且文本颜色和字体都继承父元素的颜色和字体
  • 固定不同级别标题字体大小 h1=36px, h2=30px,h3=24px,h4=18px,h5=14px,h6=12px

2.标题类名 .h1-h6 可以给所有标签应用，如：

div.h1 //则div1内的字体大小是36px；

3.副标题small代码：

bootstrap-表格

1.table的基本样式：

<table>
    <thead>
      <tr>
        <th></th>  //标题
      </tr>
    </thead>
    <tbody>
      <tr>
        <td></td> //主要内容
        <td></td>
        <td></td>
      </tr>
    </tbody>
 </table>

2.基础表格：

  • 给table表格设置了margin-bottom：20px以内设置了单元内距；
  • 在thead底部设置了一个2px的浅灰实现
  • 每个单元格顶部设置了一个1px的浅灰实现

3.附加样式表:

  • .table-striped：斑马线表格
  • .table-bordered:带边框的表格
  • .table-hover:鼠标悬停高亮的表格
  • .table-condensed:紧凑型表格

4.响应式表格:
.table-responsive响应式

<div class="table-responsive">
    <table class="table table-borderd">   //具有响应式带边框的表格
    </table>
</div>
  • Bootstrap提供了一个容器，并且此容器设置类名.table-responsive，此容器就具有响应式效果，然后将<table class="table"></table>置于这个容器中，这样表格也就具有响应式效果。
  • 当你的浏览器可视区域小于768px时，表格底部会出现水平滚动条。当你的浏览器可视区域大于768px时，表格底部水平滚动条就会消失。
