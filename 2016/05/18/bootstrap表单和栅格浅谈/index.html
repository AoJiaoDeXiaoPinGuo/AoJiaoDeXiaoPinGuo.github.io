bootstrap表单和栅格浅谈

  • bootstrap
  • 栅格

Bootstrap-表单

1.基础表单

  • 对于基础表单bootstrap并未对其做太多的定制型效果设计，仅仅对齐表单内的fieldest、legend、label标签进行了定制，主要将这些原色的margin、padding和border等进行了细化设置.

  • input、select、texture等元素设置form-control类

    • 宽度变成了100%
    • 设置了一个浅灰色（#ccc）的边框
    • 具有4px的圆角
    • 设置阴影效果，并且元素得到焦点之时，阴影和边框效果会有所变化
    • 设置了placeholder的颜色为#999

  • role

    • html5的属性
    • role=form //告诉辅助设备（如屏幕阅读器）这个元素所扮演的较色是个表单
    • role=button //告诉设备，这个事按钮，可以点击，本质上是增强语义性，增强组件的可访性、可用性
  • .form-group类名 ： 会产生margin的效果，一个margin-bottom=15px的效果

    2.form

    form-input:

  • 必须带有type，多使用form-control类，是input标签100%自适用，
    • input-sm :让控件比正常大小更小
    • input-lg :让控件比正常大小更大
    • 不设置就是正常大小
  • disabled 禁用属性
  • placeholder 默认内容

form-select:

  • 默认选择一个选项，多使用form-control
  • 多行选择的话，设置multiple属性 <select multiple class="form-control">

form-textarea:

设置rows可定义其高度，设置cols可以设置其宽度，但如果textarea元素中添加了类名form-control，则无需设置cols属性

form-fieldset/legend/label

留白

form-单选框，多选框

  • 日常开发最头痛的是控件与文字对齐问题,bootstrap则无需关心这个问题。
  • 如果checkbox需要水平排列，只需要在label标签上添加类名，checkbox-inline
  • 如果radio需要水平排列，只需要在label标签上添加类名radio-inline

    form-button

  • 在bootstrap框架中的按钮都是采用<button>来实现的
  • Bootstrap框架首先通过基础类名.btn定义了一个基础的按钮风格，然后通过.btn-default定义了一个默认的按钮风格。默认按钮的风格就是在基础按钮的风格的基础之上修改了按钮的背景颜色，边框颜色，文本颜色:
`<button class="btn btn-default">默认按钮</button>`
.btn-primary :主要按钮
.btn-success :成功按钮
.btn-warning :警告按钮
.btn-danger  :危险按钮
.btn-link    :连接按钮
.btn-lg          :大型按钮
.btn-sm          :小型按钮
.btn-xm          :超小型按钮

.btn类还支持a标签，type-submit的input标签

水平表单

在bootstrap中要实现水平表单效果，必须满足一下两个条件

1.在<form>元素是使用form-horizontal
2.配合bootstrap框架的网格系统

内联表单

  • 要实现一个美观简单的表单是轻而易举的，只要在<form>元素中添加类名form-inline即可
  • label标签会导致input换行显示，解决方案就是给label添加一个类名sr-only，标签没实现就是这个样式将标签隐藏了。
  • 如果没有为输入控件设置label标签，屏幕阅读器将无法正确识别，这也是bootstrap框架另一个优点之处，为残障人员进行了一定的考虑。

列子：

<div class="container">
    <form action="">
          <div class="form-group">
                <label for="username"> <span class="glyphicon glyphicon-user"></span>用户名</label>
                <input class="form-control" type="text" id="username" placeholder="用户名/手机号">
          </div>
          <div class="form-group">
                <label for="passowrd"> <span class="glyphicon glyphicon-user"></span>密码</label>
            <input class="form-control" placeholder="密码" type="password" id="password">
          </div>
          <div class="form-group">
            <input type="checkbox"> 记住我
          </div>
          <button class="btn-primary">提交</button>
      </form>
</div>
