github安装指南

http://25.io/mou/

安装步骤：

1.安装Node.js官网下载

2.安装Git（安装Xcode就行，本身自带git）

3.在github官网注册账号

4.安装Hexo(Hexo是npm中的一个模块)

nodeJS和git安装好后，需要创件一个文件夹，如blog,用户存放在hexo的配置文件，然后进入blog里安装Hexo

在终端进行安装,操作命令:sudo npm install -g hexo

初始化后执行hexo init的操作,命令:hexo init

现在全部的安装工作完成,blog就是你的博客根目录

启动本地服务,命令：hexo server 这样可以看到hexo默认的博客样式，根据终端提供的URL打开就行
