gulp-jade的应用

jade的应用

什么是jade？

jade是一种高性能的模板引擎，它是javascript实现的，并且可以供Node使用。

安装（本文用的是gulp-jade）

npm install gulp-jade  //[gulp的安装详见](http://aojiaodexiaopinguo.github.io/2016/02/02/gulp%E7%9A%84%E4%BD%BF%E7%94%A8/#more)

API

在gulp中的gulpfile.js中操作

var gulp=require('gulp');
var jade=require('gulp-jade');

标签的应用：

html

它会被转化为<html></html>

标签也是可以有id的：

div#header

它会被转化为<div id="header"></div>

怎么加class呢?

div.container

它会被转换为<div class="container"></div>

多个id和class也是很方便的：

div#header.container

转换为<div id="header" class="container"></div>

而当不写标签名时：

#header
.footer

转换为<div id="header"></div><div class="footer"></div>

标签文本

只要简单的把内容放在标签之后就行：

p hello world

它会被渲染为<p>hello world</p>
而当大段的文字时：

p
| hello world
| welcome to use gulp-jade
| jade makes your code pertty

渲染为<p>hello world welcome to use gulp-jade jade makes your code prtty<>

还可以用.来开始一段文字：

p.
 foo asjdfl
 asdf
  asf 
  sdf
 asfsdf
 asdf.

会被渲染为：

<p>foo asjdfl
asdf
 asf
 asf
asfsdf
asdf
.
<p>

这和带一个空格的.是不一样的，带空格的会被jade忽略，当做普通的文字：

p .

渲染为：

<p>.</p>

需要注意的时文本块需要两次转义，比如想要输出下面的文本：

<p>hello\world</p>

使用：
p.
hello\world

注释

单行注释和javascript里的一样，通过//来开始，并且必须单独一行:

// just a test
p hello
p world

渲染为：

<!--just a test-->
<p>hello</p>
<p>world</p>

jade同样支持不输出的注释，加一个短横线就行了：

//- just a test
p hello
p world

渲染为：

<p>hello</p>
<p>world</p>

内联

jade支持以自然的方式定义标签嵌套：

ul
    li.first
        a(href="#") hello
    li
        a(href="#") world
    li
        a(href="#") jade

块展开

块展开可以帮你在一行内创建嵌套的标签，下面的例子和上面的是一样的：

ul
    li.first: a(href="#") hello
    li: a(href="#") world
    li: a(href="#") jade

强大的include

include允许静态包含一段jade，或者别的存放在单个文件中的东西比如CSS，HTML非常常见的例子是包含头部和页脚，假设我们有一个下面的目录结构的文件夹：

./index.jade
./allJade
    ./head.jade

下面是index.jade的内容：

html
    head
        meta(charset="utf-8")
        title I am jade
    body
        h2 hello world
        include alljade/head.jade

head.jade的内容为：

ul
    li 
        a(href="#") 首页
    li
        a(href="#") 产品目录
    li
        a(href="#") 联系我们

则此时，include会将head.jade的内容读取到index.jade中，实现两者完美融合
渲染为：

<html>
    <head>
        <meta charset="utf-8">
        <title>I am jade</title>
    </head>
    <body>
        <h2>hello world</h2>
        <ul>
            <li>
                <a href="#">首页</a>
            </li>
            <li>
                <a href="#">产品目录</a>
            </li>
            <li>
                <a href="#">联系我们</a>
            </li>
        </ul>
    </body>
</html>

最后执行的代码

在gulpfile.js将获取的API后，输入执行代码：

gulp.task('default'.function(){
    gulp.src('alljade/*.jade')
    .pipe(jade())
    .pipe(gulp.dest('new/')) //new是一个文件夹名字,指输出结果到new这个文件夹下
})

最后在终端执行：

gulp

则会将jade格式转换为我们熟悉的html格式了。
